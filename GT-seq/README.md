
## GT-seq of *Calamagrostis* individuals collected from wild population in the Northwest Territories of Canada and Commercial *C. canadensis* indivuals.

**File organization:**

1. Raw GT-seq data and Genotypes
2. Primers (target loci primer info)
3. Scripts (modified Genotyper script for GT-seq pipeline)
4. Analysis (Complete downstream data analysis of species identity and population differentiation)

**Sequencig Details:**
GT-seq library was sequenced on an Illumina MiSeq at the KHSC using 2 × 75 bp at a loading concentration of 8 pM library and 2% PhiX.
