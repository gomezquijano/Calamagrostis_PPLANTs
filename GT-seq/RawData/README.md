## RAW Data Availability:

The raw sequencing data can be downloaded from the NCBI BioProject accession: **PRJNA1373265**

## Individual Genotypes:

the `Genotypes` directory contains .csv files with the individual locus genotypes for each sample. This is the GT-seq pipeline (https://github.com/GTseq/GTseq-Pipeline/tree/master) output which was used for all downstream abalyses.
