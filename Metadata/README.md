## Data files and column descriptions:

Sample_info_Calamagrostis.csv --> This dataset contains the geospatial location ans sequencing method of each of the 58 wild populations that were collected by the Aurora Research Institute (ARI). Eight commercial cultivar seed mixed are included, since geospatial location was unknown, the location of the commercial cultivar was included. 

Collections were conducted between 2005-2008 under the licence number 13896 on disturbed areas in the Inuvialuit Settlement Region and east of the Dempster between Inuvik and Tsiigehtchic.


### column descriptions:
Sample_info_Calamagrostis.csv  --> For each seed source there is the following information: species name, origin (wild/commercial),latitude and longitude, seed source name, a two letter geogrpahical region code, region name, number of individuals from each seed source,sequencing method (ddRAD/GT-seq).
