## ddRAD of *Calamagrostis* individuals collected from wild population in the Northwest Territories of Canada 

#### File organization:

  1. Raw ddRAD-seq data
  2. Analysis pipeline
  3. SNP Calling Scripts uisng STACKS

#### Sequencig Details: 

ddRAD library was sequenced at Génome Québec, Canada in an Illumina HiSeq 4000 partial flow cell with 2 x 150 bp paired end reads at a loading concentration of 360 pM library, producing a total of  210,819,849 raw reads.
